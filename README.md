# Project Info.

Created a MERN stack food ordering application using React.

1. ReactJs | HTML|CSS|JavaScript | NodeJs | ExpressJs | Bootstrap | MongoDB
2. Tools: Visual Studio code, Post-man
3. GoFood is a user-friendly, responsive food ordering application built with ReactJS, offering seamless features including user authentication (Sign up and login), uses Hashing for password encryption using bcrypt.js, cart management, checkout, and order history tracking.
4. The application leverages MongoDB to store user data, login credentials, and product information. Data retrieval is managed through ExpressJS APIs, ensuring efficient communication between the back-end and front-end components.

## Available Scripts

In the project directory, you can run:

### `npm install`

This will get and install all the required dependencies for React.

### `npm start`

Runs the app in the development mode.\
Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

The page will reload when you make changes.\
